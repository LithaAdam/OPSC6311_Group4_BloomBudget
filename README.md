# OPSC6311_Group4_BloomBudget
Repository for group 4 BloomBudget

The POE required us as students to work together as a group to create a fully operational Android
application that is a budget planner assistant. This application is meant to help users plan and
manage their finances whether they are saving for a specific item or purchase, or are merely saving
money just because. The application we created is called BloomBudget. BloomBudget should:
* Help users track their spending habits
* Track their saving goals
* Provide some gamification for user amusement

This project consisted of 3 Parts, Part 1 mainly focusing on research, planning and design. Part 2 was
about building a prototype for BloomBudget. The final POE required a fully functional application with
all the necessary features and additional features.
